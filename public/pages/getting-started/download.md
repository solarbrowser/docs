# Download & Installation

A straightforward path to installing Solar. This section covers how to download and run the browser with no hidden steps or unnecessary decisions. One build one flow and a clean start designed to work immediately and stay out of your way.

## Choose your platform

Solar is available across major desktop and mobile platforms. Each build is crafted specifically for its environment ensuring consistent behavior performance and stability without compromise. Choose your platform and download a version designed to feel native light and precise.

When we are ready to release Solar you will be able to download the version that belongs to your operating system from [here](https://browser.solar/download)


## System requirements

Solar is built to run efficiently without demanding excess resources. The requirements are intentionally modest allowing the browser to stay responsive on modern systems while preserving its clean architectural boundaries. At launch we plan to support a wide range of systems including many older devices & operating systems though final limits will be defined closer to release as the platform matures. No artificial limits no inflated expectations.

## Installation steps

Installation is intentionally simple. Download run and you are done. There are no bundled components no silent background services and no hidden configuration steps. What you install is exactly what runs.

## First run

On first launch Solar stays quiet. No accounts no forced setup no interruptions. The browser opens ready to use offering a clean space where interaction feels immediate and unencumbered.
